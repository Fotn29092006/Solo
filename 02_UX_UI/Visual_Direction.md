## Content
Mood:

- Elite.
- Mysterious.
- Progression-driven.
- Powerful.
- Game-like but clean.

Use:

- Dark UI.
- High contrast.
- Neon cyan or electric accent highlights.
- Cyber HUD feel.
- Rank badges.
- XP bars.
- Glowing but controlled cards.
- System overlays.
- Futuristic typography treatment.

Mobile principles:

- First screen should be useful, not a marketing landing page.
- Logging flows must be quick.
- Text must remain readable in Telegram WebView.
- Motion should clarify state changes, not slow the user down.
