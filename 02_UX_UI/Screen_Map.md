## Content
Main MVP screens:

- Home: system status, level, rank, XP, today quests, streaks, quick logs.
- Character: avatar, stats, path, titles, domain levels.
- Quests: daily quests, weekly quests, completed and failed states, rewards.
- Body: weight, metrics, workout summary, body domain status.
- Nutrition: meal log, water log, macros, compliance.
- Mind: learning sessions, focus history, knowledge stats.
- Language: session log, streak, skill categories, weekly total.
- Weekly Review: weigh-in, reflection, score, system analysis.
- Settings: goals, notifications, paths, Telegram settings, privacy.

MVP navigation should stay compact and mobile-first.
