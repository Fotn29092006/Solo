# Solo System RPG Design Skill

## Visual Direction

Design for a premium, dark, system-like Telegram Mini App inspired by Solo Leveling progression screens.

Use:

- High contrast dark surfaces.
- Cyan or electric accent highlights.
- Strong hierarchy.
- Compact mobile flows.
- XP bars, rank badges, quest states, and system messages.
- Smooth but purposeful motion.

Avoid:

- Desktop-first layouts.
- Bloated navigation.
- Decorative stats that do not map to real logic.
- Generic fitness tracker styling.
- Visual polish that hides weak product mechanics.

1. Read `00_START_HERE/06_Graph_Memory_Protocol.md`.
