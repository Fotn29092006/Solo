## Content
Daily loop:

1. User receives or opens daily quest package.
2. User logs actions quickly in the Mini App.
3. System evaluates completion and quality.
4. User receives XP, streak, stat, or rank feedback.
5. Telegram reminders protect the loop when needed.

Weekly loop:

1. User submits weekly check-in.
2. System reviews weight, adherence, energy, sleep, stress, wins, and obstacles.
3. System updates scores and mission difficulty.
4. User receives a system-like weekly summary and next-week direction.
